### Examples

*Standard Use*

```
<FitnessLeaderboard
  campaign='au-21638'
/>
```

*Leaderboard by Team*

```
<FitnessLeaderboard
  campaign='au-21937'

/>
```

*Leaderboard by Group*

```

  campaign='au-21937'
  type='group'
  groupID={58}
/>
```

*Leaderboard by Elevation*

```
<FitnessLeaderboard
  campaign='nz-1608'
  sortBy='elevation'
/>
```
